\hypertarget{index_autotoc_md1}{}\doxysection{\texorpdfstring{Overview}{Overview}}\label{index_autotoc_md1}
This project is a C++ application that converts La\+TeX documents into Markdown format. It uses Flex and Bison for lexical analysis and parsing. The parser processes various La\+TeX constructs, such as sections, lists, tables, and more, and converts them into equivalent Markdown representations.\hypertarget{index_autotoc_md2}{}\doxysection{\texorpdfstring{Features}{Features}}\label{index_autotoc_md2}

\begin{DoxyItemize}
\item Convert La\+TeX sections and subsections to Markdown headers.
\item Handle ordered and unordered lists.
\item Support for tables, figures, and verbatim text.
\item Conversion of La\+TeX formatting (bold, italic) to Markdown.
\item Output Markdown to a file.
\end{DoxyItemize}\hypertarget{index_autotoc_md3}{}\doxysection{\texorpdfstring{Project Structure}{Project Structure}}\label{index_autotoc_md3}
